# Azure Machine Learning (microsoft-azure-machine-learning)
Azure Machine Learning REST API provides management of ML workspaces, compute resources, datasets, experiments, models, and endpoints. It supports the full ML lifecycle including data preparation, model training, evaluation, deployment, and monitoring.
